Add a Recipe

The best way to add a recipe is to send it to Claude — you can share a link, paste a screenshot, or type it out. Claude will add it to the book and give you an updated file to upload to GitHub.

1
Open Claude and copy this prompt
Go to claude.ai, open a new chat, and paste the prompt below. Then attach your screenshot or paste your link underneath it.
I'd like to add a new recipe to our shared recipe book. I'll attach a screenshot or paste a link below. Please: 1. Read the full recipe from the screenshot or link 2. Ask me which group it belongs to (Frits or Shiela) and which category (Breakfast, Lunch, Dinner, Snacks, Baking, Desserts, Drinks, or Other) 3. Add it to the existing recipe-book.html file — keeping all existing recipes intact — and give me the updated file to download 4. Make sure ingredients section headers (lines starting with "For the...", "Sauce:", "Toppings:" etc.) are preserved as-is so they render correctly The recipe book is a standalone HTML file hosted on GitHub Pages.
2
Attach your recipe
In the same Claude message, either:
  • Screenshot: tap the attachment icon and upload a photo or screenshot of the recipe
  • Link: paste the URL of the recipe page
  • Type it out: paste the recipe text directly into the message
3
Download the updated file and upload to GitHub
Claude will give you an updated recipe-book.html file with your new recipe already included. Then:
  1. Save the file as index.html
  2. Go to your GitHub repository
  3. Click Add file → Upload files
  4. Upload the new file — it will overwrite the old one
  5. Click Commit changes — the site updates within a minute
or add manually

Manually added recipes are only saved on this device. Use the Claude method above to add permanently for everyone.
